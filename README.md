# Dev UI Demo

This is a .NET 9 app which use the latest .NET DEV UI package.

## How to run in GitHub Codespaces

1. Open the repo in codespaces.
2. Open the terminal.
3. Run the command `dotnet run`.
4. Modify the forwarded URL and append `/devui` view the Dev UI.

## How to run in local machine.

1. Clone the repo.
2. Set an the GitHub Token as user secret using `dotnet user-secrets set GITHUB_TOKEN github_pat_YOUR_GITHUB_PAT`.
3. Run the command `dotnet run`.
4. Modify the forwarded URL and append `/devui` view the Dev UI.

When running in GitHub Codespaces you don't need to set the GITHUB_TOKEN environment variable.
